Introducción

MvFy - HSV es un sistema de reconocimiento facial que integra Face-api.js, como sistema principal en la detecciónn y el reconocimiento de rostros, este modulo tiene el objetivo de proporcionar herramientas sencillas de integracion en un sistema cerrado de camaras.

Inicio rapido

Para empezar a usar el proyecto se recomienda ir al repositorio de Face-recognition , y clonar dicho repositorio.

git clone git@github.com:erwingforerocastro/Face-Recognition.git

  • Para integrar con el proyecto javascript se debe hacer uso de la importacion modular de la clase principal
    • import { MvfyHsv } from 'ruta-relativa/mvfy-hsv'
    const MvFyHsv = require('ruta-relativa/mvfy-hsv')
    Donde ruta-relativa es dicha ruta donde esta posicionado el archivodel proyecto Node.js

  • para iniciar el sistema, se debe pasar los siguientes parametros, los cuales permitiran comenzar la detección de los usuarios en un sistema cerrado por medio de un video RGB

    • name name_file video_input min_date_knowledge file_extension features type_system
    const MODEL = MvfyHsv({ video_label: 'video_demo', min_date_knowledge: '1 day', file_extension: 'xlsx', features: 'all', type_system: 'optimized'});
